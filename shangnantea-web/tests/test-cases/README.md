# 测试用例

## 文档说明

本目录存放所有测试用例文件。

**创建日期**: 2024-12-17

---

## 目录结构

```
test-cases/
├── user/          # 用户模块测试用例
├── tea/            # 茶叶模块测试用例
├── shop/           # 店铺模块测试用例
├── order/          # 订单模块测试用例
├── forum/          # 论坛模块测试用例
└── message/        # 消息模块测试用例
```

---

## 测试用例文件命名规范

- 格式：`[功能名].spec.js`
- 示例：`address.spec.js`、`favorites.spec.js`

---

## 测试用例编写

测试用例必须依照 `test-case-templates/` 目录下的模板编写。
