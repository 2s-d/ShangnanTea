## 使用方法

1. 启动项目：`npm run serve`
2. 访问：`http://localhost:8080/test/dataflow`
3. 选择"用户模块"标签页
4. 点击具体API测试按钮或"运行所有用户测试"

## 特点

- ✅ **真实环境测试**：直接调用Vuex store actions
